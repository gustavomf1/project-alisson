**O Projeto**
*O projeto se trata de uma aplicação que recebe categorias e produtos, onde cada produto precisa ter uma categoria*
*É possível ver a categoria a qual o produto pertence clicando sobre o produto em questão*

**Instruções**
*O arquivo compactado deve ser extraído direto na pasta htdocs (do xampp) para que as rotas funcionem corretamente*

**Rotas**
*Categoria*
*Criar: http://localhost/phpfaculdade/aula_php/aula7/categorias/criar*
*Ver: http://localhost/phpfaculdade/aula_php/aula7/categorias/ver?id=1*
*Editar: http://localhost/phpfaculdade/aula_php/aula7/categorias/editar?id=2

*Produto*
*Criar: http://localhost/phpfaculdade/aula_php/aula7/produtos/criar*
*Ver: http://localhost/phpfaculdade/aula_php/aula7/produtos/exibir?id=1*
*Editar: http://localhost/phpfaculdade/aula_php/aula7/produtos/editar?id=2

*Lembrando que o excluir e o editar de ambos está disponível através de um botão*

Baixar código	Pega seu repositório do GitHub
Instalar PHP	Usa o PHP 8.2 com suporte a MySQL
Subir MySQL	Cria um banco de dados de teste temporário
Rodar Composer	Instala dependências lá no ambiente (você não precisa instalar nada local)
Testar conexão	Testa se o PHP consegue conectar no MySQL
Rodar teste simples	Executa um pequeno teste de PHP pra confirmar que o workflow funciona
